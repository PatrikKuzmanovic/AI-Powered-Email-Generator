# AI-Powered Email Generator

A sleek, responsive, single-page web application that leverages the Gemini API to automatically draft professional emails based on user context. Built as an AI portfolio project.

## Features
- **Context-Aware Generation:** Generates emails customized by type (cold outreach, follow-up, customer support, etc.), tone, and length.
- **Real-Time AI Processing:** Integrates seamlessly with the powerful Gemini Flash model.
- **Copy & Edit Functionality:** Effortlessly copy to clipboard or tweak the AI's response directly in the output window before sending.
- **Responsive Design:** Clean, modern, two-column UI that works perfectly on desktop and mobile.

## Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom design system without external CSS frameworks for a tailored, professional look.
- **Vanilla JavaScript (ES6):** Modularized for UI handling and API querying.
- **Gemini API:** For intelligent content generation.

## Setup & Running Locally
Since this is a client-side vanilla web application, no build tools or servers are strictly required!

1. Clone or download this repository.
2. Open `js/api.js` and ensure your Gemini API Key is set in the `this.apiKey` variable.
3. Open `index.html` directly in your web browser, or use a tool like VS Code Live Server.
